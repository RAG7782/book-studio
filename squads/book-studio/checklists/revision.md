# Checklist de Revisao

Verificar antes de enviar o manuscrito para formatacao.

## Revisao Estrutural

- [ ] Arco narrativo completo e satisfatorio
- [ ] Todos os subplots resolvidos
- [ ] Pacing equilibrado (sem sags)
- [ ] Midpoint efetivo
- [ ] Climax preparado e impactante
- [ ] Resolucao fecha pontas abertas

## Revisao de Consistencia

- [ ] Nomes grafados consistentemente
- [ ] Timeline sem contradicoes
- [ ] Descricoes fisicas consistentes
- [ ] Geografia e distancias coerentes
- [ ] Regras do mundo respeitadas
- [ ] Fatos nao-contraditorios

## Copy Editing

- [ ] Gramatica verificada
- [ ] Ortografia verificada
- [ ] Pontuacao consistente
- [ ] Dialogos formatados corretamente (travessao)
- [ ] Repeticoes eliminadas
- [ ] Voz passiva minimizada
- [ ] Cliches substituidos

## Qualidade

- [ ] Analise critica realizada (Critic)
- [ ] Beta reading simulado
- [ ] Feedback incorporado
- [ ] Pontos fracos identificados e corrigidos
- [ ] Autor satisfeito com o resultado

## Completude

- [ ] Todos os capitulos escritos e revisados
- [ ] Front matter pronto (dedicatoria, epigrafe)
- [ ] Back matter pronto (agradecimentos, sobre o autor)
- [ ] Bibliografia/referencias (se aplicavel)
- [ ] Notas de rodape verificadas (se aplicavel)
