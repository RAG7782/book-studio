# Checklist de Publicacao

Verificar antes de distribuir o livro nas plataformas.

## Manuscrito

- [ ] Manuscrito final aprovado pelo autor
- [ ] Todas as revisoes incorporadas
- [ ] Proofreading final concluido
- [ ] Versao final salva e versionada

## Formatacao

- [ ] Markdown estruturado e limpo
- [ ] EPUB gerado e testado em e-reader
- [ ] PDF (se aplicavel) revisado
- [ ] Tipografia consistente
- [ ] Table of Contents funcional
- [ ] Links internos funcionando (EPUB)

## Capa

- [ ] Capa criada (AI ou designer)
- [ ] Dimensoes corretas por plataforma
- [ ] Titulo e autor legiveis em thumbnail
- [ ] Genero comunicado visualmente
- [ ] Versoes: e-book, paperback (se aplicavel)

## Metadados

- [ ] Titulo e subtitulo finais
- [ ] Categorias BISAC selecionadas (2-3)
- [ ] Keywords SEO definidas (7)
- [ ] Sinopse escrita e revisada
- [ ] Bio do autor pronta
- [ ] Faixa etaria classificada
- [ ] ISBN obtido (se necessario)
- [ ] Preco definido por formato

## Marketing

- [ ] Press release escrito
- [ ] Posts para redes sociais prontos
- [ ] Email de lancamento criado
- [ ] Landing page (se aplicavel)
- [ ] ARC enviados para reviewers
- [ ] Data de lancamento definida

## Plataformas

- [ ] Amazon KDP configurado
- [ ] Google Play Books (opcional)
- [ ] Kobo (opcional)
- [ ] Apple Books (opcional)
- [ ] Impressao sob demanda (opcional)

## Pos-lancamento

- [ ] Monitorar reviews nos primeiros 7 dias
- [ ] Responder feedback construtivo
- [ ] Ajustar keywords se necessario
- [ ] Primeira promocao planejada
