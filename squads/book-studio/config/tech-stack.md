# Tech Stack — Book Studio

## Formato Principal
- **Markdown** — Formato fonte de todos os manuscritos
- **YAML** — Metadados e configuracoes

## Ferramentas de Conversao
- **Pandoc** — Conversao MD → EPUB, PDF, DOCX, HTML
- **Calibre** — Gestao e conversao de e-books
- **WeasyPrint** — MD → PDF com CSS (alternativa)

## AI & LLM
- **Claude** — Agentes do squad (via AIOS)
- **Groq** — Geração rapida de rascunhos
- **OpenRouter** — Backup/alternativa

## Geracao de Imagens (Capas)
- **DALL-E 3** / **Midjourney** — Via prompts do Formatter
- **Canva** — Layout de capa e material de marketing

## Distribuicao
- **Amazon KDP** — Principal canal digital e impresso
- **Google Play Books** — Android
- **Kobo** — Internacional
- **Draft2Digital** — Distribuicao agregada

## Versionamento
- **Git** — Controle de versao do manuscrito
- **GitHub** — Backup remoto (repositorio privado)

## Formatacao
- **VS Code** — Editor de Markdown
- **Typora** — Preview WYSIWYG (opcional)
- **Obsidian** — Notas e worldbuilding interligados
