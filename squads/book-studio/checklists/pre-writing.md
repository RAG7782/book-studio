# Checklist de Pre-escrita

Verificar antes de comecar a escrever o primeiro capitulo.

## Conceito

- [ ] Premissa definida e aprovada (Muse)
- [ ] Logline clara e concisa
- [ ] Publico-alvo identificado
- [ ] Genero e tom definidos
- [ ] Comp titles identificados

## Personagens (Ficcao)

- [ ] Protagonista com ficha completa
- [ ] Antagonista com motivacao clara
- [ ] Personagens secundarios esbo√ßados
- [ ] Arcos de transformacao planejados
- [ ] Vozes distintas definidas

## Pesquisa

- [ ] Pesquisa tematica concluida (Researcher)
- [ ] Worldbuilding documentado (se ficcao)
- [ ] Fontes organizadas (se nao-ficcao)
- [ ] Lacunas de pesquisa identificadas e marcadas

## Estrutura

- [ ] Outline completo (Architect)
- [ ] Beats por capitulo definidos
- [ ] Timeline de eventos consistente
- [ ] Pacing mapeado
- [ ] Ponto de virada central definido

## Preparacao

- [ ] Diretorio do projeto criado
- [ ] Template de capitulo disponivel
- [ ] Guia de estilo definido (dialogo, formatacao)
- [ ] Meta de palavras por capitulo definida
- [ ] Rotina de escrita planejada
